Gemini Live TTS v2.4

Turn text and images into natural-sounding speech using Google's Gemini AI with 30 HD voices.

Step 1: Set Up Your API Key

Before you can use the extension, you'll need a Gemini API key:

  1. Go to Google AI Studio and click "Create API key".
  2. Copy the key into the extension's settings page.
  3. Choose your preferred API type, model, and voice.
  4. Press "Save".
Note: Keep your API key private and don't share it with others.

Step 2: Choose Your API Type

This extension supports two different Gemini APIs:

Native Audio (Live API) LIVE

Real-time WebSocket connection. Supports images. Natural voices with emotions. Best for conversations and image descriptions.

Text-to-Speech TTS

REST API. Precise text reading. Cheaper pricing. Best for audiobooks, articles, and exact pronunciation.

How to Use

Text to Speech:

Select any text on a webpage, right-click, and choose "Transcribe with Gemini" from the context menu.

Image to Speech (Native Audio only):

Click the extension icon in your address bar to activate the image selection tool. Click and drag to select an area.

Playback Controls:

Click the extension icon in your toolbar to open the playback control popup with play/pause, stop, and volume controls.
